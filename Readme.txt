This project is created to make a “To Do List” for users who can write here what they need to do through the day. All the data is saved at ATLAS MongoDB which is currently on free pack.

This project consist of 
HTML, CSS, BOOTSTRAP, JAVASCRIPT, NODEJS, EXPRESS JS, MONGO DB

These all language make this project more amazing in functionalities.
